# 分布式追踪

## 一、概念

>分析和监控应用的监控方法
>查找故障点、分析性能问题
>起源于 Google 的 Dapper
>OpenTracing：
>
>> API 规范、框架、库的组合

![image-20250610172903099](image-20250610172903099.png)

## 二、什么是Jaeger？

>开源、端到端的分布式追踪系统
>
>针对复杂的分布式系统，对业务链路进行监控和问题排查

![image-20250610173055682](image-20250610173055682.png)

## 三、专业术语

![image-20250610173230247](image-20250610173230247.png)

### 1、Span

>逻辑单元
>
>有操作名、执行时间
>
>嵌套、有序、因果关系

### 2、Trace

>数据/执行路径
>
>Span 的组合
