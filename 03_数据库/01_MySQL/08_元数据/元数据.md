# information_schema 元数据获取

## 一、介绍

每次数据库启动，都会在内存中生成I_S，生成查询MySQL部分元数据信息视图

### 视图

```mysql
select语句的执行方法。不保存数据本身。I_S中的视图，保存的就是查询元数据的方法。

创建视图：
create view v_select as
select student.sno as 学号,
student.sname as 学生姓名,
count(sc.cno) as 学习课数 
from sc 
join student
on sc.sno=student.sno 
where student.sname='zhang3' 
group by student.sno;

使用：
select * from v_select;
```



## 二、I_S.tables

### 1、介绍

```mysql
作用：保存了所有表的数据字典的信息
 use information_schema
 show tables;
 ...
 TABLES 
 ...
 desc tables;
+-----------------+---------------------+------+-----+---------+-------+
| Field           | Type                | Null | Key | Default | Extra |
+-----------------+---------------------+------+-----+---------+-------+
 ...
| TABLE_SCHEMA		表所在的库
| TABLE_NAME		表名
| ENGINE			引擎
| TABLE_ROWS		表的数据行（不是特别实时）
| AVG_ROW_LENGTH	平均行长度
| DATA_LENGTH		表使用存储空间大小
| INDEX_LENGTH      表中索引占用空间大小
| DATA_FREE         表的碎片情况
...
```



### 2、企业应用案例

#### 例子一：数据库资产统计，统计每个库下所有表的个数，表名

```mysql
mysql> select table_schema, 
count(table_name), 
group_concat(table_name) 
from information_schema.tables 
group by table_schema;
```



#### 例子二：统计每个库占用总空间的大小（结果不一定实时）

```mysql
方法一
#一张表的大小公式 = TABLE_ROWS*AVG_ROW_LENGTH+INDEX_LENGTH （推荐）
select table_schema,
sum(avg_row_length*table_rows+index_length)
from information_schema.tables
group by table_schema;

方法二
#一张表的大小公式 = DATA_LENGTH
select table_schema,
sum(data_length)
from information_schema.tables
group by table_schema;

注意：以上两种搜索的结果大小是字节单位
select table_schema,
sum(avg_row_length*table_rows+index_length)/1024  #KB
from information_schema.tables
group by table_schema;


select table_schema,
sum(avg_row_length*table_rows+index_length)/1024/1024  #MB
from information_schema.tables
group by table_schema;
方法二同理
```



#### 例子三：查询业务数据库（系统库）中，所有非InnoDB表。

```mysql
select table_schema,
engine,
table_name from information_schema.tables
where engine != 'InnoDB'
and table_schema not in ('sys','performance_schema','information_schema','mysql');
```



#### 例子四：例子三：查询业务数据库（系统库）中，所有非InnoDB表。将非InnoDB表转化为InnoDB表

```mysql
#使用concat拼接，先打印出想要的结果

from information_schema.tables
where engine != 'InnoDB'
and table_schema not in ('sys', 'performance_schema','information_schema','mysql');
```
