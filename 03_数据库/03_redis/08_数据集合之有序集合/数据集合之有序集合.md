# sortset有序集合

## 一、特点

```bash
key     value 
stu     100, 20, 10 
        (ls, pwd, mkdir) 
         0    1     2 
         
ps：有序数据类型都有下标索引
```



## 二、应用场景

```bash
排行榜应用
```



## 三、使用

### 1、ZADD：添加数据

```bash
127.0.0.1:6379> ZADD myzety 1 one
1
127.0.0.1:6379> ZADD myzety 2 two
1
127.0.0.1:6379> ZADD myzety 20 twoth  #twenty,打错了
1
127.0.0.1:6379> ZADD myzety 3 three
1
```



### 2、查询数据

#### 1.ZRANGE：查看指定区间数据数据

**查看所有数据**

```bash
127.0.0.1:6379> ZRANGE myzety 0 -1
one
two
three
twoth
127.0.0.1:6379> 
```

**查看指定区间数据**

```bash
127.0.0.1:6379> ZRANGE myzety 0 2
one
two
three
127.0.0.1:6379> ZRANGE myzety 0 1
one
two
127.0.0.1:6379> 

```



#### 2.ZCARD：查看数据的个数

```bash
127.0.0.1:6379> ZCARD myzety
4
```



#### 3.ZCOUNT：获取指定分数区间内的数据个数

```bash
127.0.0.1:6379> ZCOUNT myzety 1 40
4
127.0.0.1:6379> ZCOUNT myzety 2 40
3
```



#### 4.ZLEXCOUNT：根据字典排序返回min ,max之间的数据个数

```bash
127.0.0.1:6379> ZLEXCOUNT myzety [a [z
4
127.0.0.1:6379> ZLEXCOUNT myzety [ti [z
3
```



#### 5.ZSCORE：获得指定数据的分数值

```bash
127.0.0.1:6379> ZSCORE myzety three
3
127.0.0.1:6379> ZSCORE myzety twoth
20
```



#### 6.ZSCAN：模糊查询

```bash
127.0.0.1:6379> ZSCAN myzety 1 match "th*"
0
three
3
```



### 3、删除数据

#### 1.ZREM：删除指定数据

```bash
127.0.0.1:6379> ZRANGE myzety 0 -1
one
two
three
twoth
127.0.0.1:6379> ZREM myzety three
1
127.0.0.1:6379> ZRANGE myzety 0 -1
one
two
twoth
```
