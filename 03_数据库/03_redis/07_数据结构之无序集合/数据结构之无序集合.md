# set无序集合

## 一、特点

```bash
key        value
stu     (zha3,ls,w5,m6)
```



## 二、应用场景

```bash
社交类的平台，好友系统  共同好友，二度好友
```



## 三、使用

### 1、SADD：增加数据

```bash
127.0.0.1:6379> SADD num 1
1
127.0.0.1:6379> SADD num 12 123 1234 12345 123456 1234567
6
```



### 2、SMEMBERS：查看

```bash
127.0.0.1:6379> SMEMBERS num
1
12
123
1234
12345
123456
1234567
```



### 3、SREM：删除

```bash
127.0.0.1:6379> SMEMBERS num
1
12
123
1234
12345
123456
1234567
127.0.0.1:6379> SREM num 1234
1
127.0.0.1:6379> SMEMBERS num
1
12
123
12345
123456
1234567
127.0.0.1:6379> 
```



### 4、SMOVE：移动数据

```bash
127.0.0.1:6379> SMEMBERS num
1
12
123
12345
123456
1234567
127.0.0.1:6379> SADD myset 1
1
127.0.0.1:6379> SMEMBERS myset
1
127.0.0.1:6379> SMOVE num myset 12345
1
127.0.0.1:6379> SMEMBERS num
1
12
123
123456
1234567
127.0.0.1:6379> 
1
12345
```

### 5、其他查看



#### 1.SCARD：返回集合当中的个数

```bash
127.0.0.1:6379> SMEMBERS num
1
12
123
123456
1234567
127.0.0.1:6379> SMEMBERS myset
1
12345
127.0.0.1:6379> SCARD myset
2
127.0.0.1:6379> SCARD num
5
```



#### 2.SRANDMEMBER：随机返回数据（抽奖）

```bash
127.0.0.1:6379> SRANDMEMBER num 2 #随机返回两个
1
1234567
127.0.0.1:6379> SRANDMEMBER num 3 #随机返回三个
123
1234567
123456
```



#### 3.SISMEMBER：判断一个数据是否在一个集合中

```bash
127.0.0.1:6379> SMEMBERS num
1
12
123
123456
1234567
127.0.0.1:6379> SISMEMBER num 123
1
127.0.0.1:6379> SISMEMBER num abc
0
```



### 6、SPOP：随机消费数据

```bash
127.0.0.1:6379> SMEMBERS num
1
12
123
123456
1234567
127.0.0.1:6379> SPOP num 1	#随机消费一个
1234567
127.0.0.1:6379> SPOP num 2  #随机消费两个
1
123456
127.0.0.1:6379> SMEMBERS num
12
123
127.0.0.1:6379> 
```



### 7、集合

**数据准备**

```bash
127.0.0.1:6379> sadd myseta 1 2 3 4 5
5
127.0.0.1:6379> sadd mysetb 4 5 6 7 8
5
127.0.0.1:6379> sadd mysetc 5 6 7 8 9
5
```



#### 1.SINTER：返回多个集合当中的交集

```bash
127.0.0.1:6379> SINTER myseta mysetb  mysetc
5
```



#### 2.SUNION：返回多个集合当中的并集

```bash
127.0.0.1:6379> SUNION myseta mysetb  mysetc
1
2
3
4
5
6
7
8
9
```



#### 3.SDIFF：返回多个集合的差集

>取多个集合的差集，以最左边的为主集合，返回左集合中有而其他集合没有的成员。

```bash
127.0.0.1:6379> SDIFF myseta mysetb  mysetc
1
2
3
127.0.0.1:6379> SDIFF mysetc mysetb myseta
9
```



#### 4.SDIFFSTORE：将多个集合的差集保存为一个新的集合

```bash
127.0.0.1:6379> SDIFF myseta mysetb  mysetc
1
2
3
127.0.0.1:6379> SDIFFSTORE mysetd myseta mysetb  mysetc
3
127.0.0.1:6379> SMEMBERS mysetd
1
2
3
```
