# 数据结构之list列表

## 一、特点

```bash
key 	     	value
day    [day7,day6,day5,day4,day3,day2,day1]
		0     1     2   3    4    5    6
		                              -1
```

> 反向列表, 最新的值在最前⾯, 列表会⾃动给列表对象值加索引



## 二、应用场景

```bash
微信朋友圈 即时消息展示
```



## 三、使用

### 1、LPUSH：从左边插入数据

```bash
127.0.0.1:6379> LPUSH people xiaowu
1
127.0.0.1:6379> LPUSH people man 22 shanghai
4
```



**LRANGE：查看数据内容**

```bash
127.0.0.1:6379> LRANGE people 0 -1	# 查看第0个到最后一个数据
shanghai
22
man
xiaowu
127.0.0.1:6379> LRANGE people 0 2	# 查看第0个到第二个数据
shanghai
22
man
```



### 3、RPUSH：从右侧插入数据

```bash
127.0.0.1:6379> LRANGE people 0 -1
shanghai
22
man
xiaowu
127.0.0.1:6379> RPUSH people linyi
5
127.0.0.1:6379> LRANGE people 0 -1
shanghai
22
man
xiaowu
linyi
```

### 4、两侧插入数据

#### 1.LPUSHX：key存在时在左侧添加键值

> key存在

```bash
127.0.0.1:6379> LRANGE people 0 -1
shanghai
22
man
xiaowu
linyi
127.0.0.1:6379> LPUSHX people 1
6
127.0.0.1:6379> LRANGE people 0 -1
1
shanghai
22
man
xiaowu
linyi
```

> key不存在

```bash
127.0.0.1:6379> LRANGE test 0 -1

127.0.0.1:6379> LPUSHX test 1
0
127.0.0.1:6379> LRANGE test 0 -1

127.0.0.1:6379> 
```





#### 2.RPUSHX：key存在时右侧添加键值

> key存在

127.0.0.1:6379> RPUSHX test aaa
0
127.0.0.1:6379> LRANGE test 0 -1

### 7、查询数据

#### 1.LLEN：查看列表中元素的个数

```bash
127.0.0.1:6379> LRANGE people 0 -1
qingpu
666
22
aaa
man
xiaowu
linyi
127.0.0.1:6379> LLEN people
7
```



#### 2.LRANGE：根据起止下标查看元素

```bash
127.0.0.1:6379> LRANGE people 0 -1
qingpu
666
22
aaa
man
xiaowu
linyi
127.0.0.1:6379> LRANGE people -1 -1		#查看最右边的元素
linyi
127.0.0.1:6379> LRANGE people 0 0		#查看最左边的元素
qingpu
127.0.0.1:6379> 

```



#### 3.LINDEX：根据指定下标查找数据

```bash
127.0.0.1:6379> LRANGE people 0 -1
666
22
aaa
man
xiaowu
linyi
127.0.0.1:6379> LINDEX people 3
man
127.0.0.1:6379> 
```



#### 4.BLPOP：实时查询列表最左数据

> 查询列表最左数据，后面需要设置数据等待时间。如果列表有数据直接返回最左数据。如果没有数据，等待指定时间，指定时间内有新数据插入则返回最新数据，如果没有数据。时间超时后退出。

```bash
127.0.0.1:6379> BLPOP people 10
people
qingpu
127.0.0.1:6379> 
```



### 8、数据消费

#### 1.LPOP：从左侧消费数据，消费后删除

```bash
127.0.0.1:6379> LRANGE people 0 -1
666
22
aaa
man
xiaowu
linyi
127.0.0.1:6379> LPOP people
666
127.0.0.1:6379> LRANGE people 0 -1
22
aaa
man
xiaowu
linyi
127.0.0.1:6379> 
```



#### 2.RPOP：从右侧消费数据，消费后删除

```bash
127.0.0.1:6379> LRANGE people 0 -1
22
aaa
man
xiaowu
linyi
127.0.0.1:6379> RPOP people
linyi
127.0.0.1:6379> LRANGE people 0 -1
22
aaa
man
xiaowu
127.0.0.1:6379> 
```



#### 3.RPOPLPUSH：消费A列表最右边数据，然后重新追加到列表B左边

```bash
127.0.0.1:6379> LRANGE people 0 -1
22
aaa
man
xiaowu
127.0.0.1:6379> LRANGE num 0 -1
5
4
3
2
1
127.0.0.1:6379> RPOPLPUSH num people
1
127.0.0.1:6379> LRANGE num 0 -1
5
4
3
2
127.0.0.1:6379> LRANGE people 0 -1
1
22
aaa
man
xiaowu
127.0.0.1:6379> 
```

### 9、数据删除

#### 1.LREM：删除指定的多个键值

```bash
127.0.0.1:6379> LRANGE people 0 -1
1
22
aaa
man
xiaowu
127.0.0.1:6379> LREM people 1 22	#删除一个22
1
127.0.0.1:6379> LRANGE people 0 -1
1
aaa
man
xiaowu
127.0.0.1:6379> 
```

**指定删除多个键时，如果列表指定键值小于要删除的键值。则只删除已有的所有指定键值**

```bash
127.0.0.1:6379> LRANGE people 0 -1
1
aaa
man
xiaowu
127.0.0.1:6379> LREM people 2 man
1
127.0.0.1:6379> LRANGE people 0 -1
1
aaa
xiaowu
127.0.0.1:6379> 
```



#### 2.DEL：删除key及所有数据

```bash
127.0.0.1:6379> LRANGE num 0 -1
5
2
127.0.0.1:6379> del num
1
127.0.0.1:6379> LRANGE num 0 -1

127.0.0.1:6379> 
```
