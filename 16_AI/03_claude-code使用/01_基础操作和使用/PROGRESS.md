## 使用说明

### 如何更新进度

1. **开始任务时**:
   - 在"当前状态 > 正在进行的任务"中添加任务
   - 更新对应阶段的任务状态为"进行中 🚧"

2. **完成任务时**:
   - 在对应阶段的任务清单中标记 ✅
   - 在"证据链接"中添加证明文件、测试结果、截图等
   - 在"完成任务记录"中添加详细记录
   - 更新"当前状态"中的任务列表
   - 更新阶段进度百分比

3. **遇到问题时**:
   - 在"问题与风险记录"中记录问题
   - 在"当前状态 > 遇到的问题"中更新

4. **重要决策时**:
   - 在"关键决策记录"中记录决策内容和原因

### 证据链接格式

证据链接应包含但不限于:
- 源代码文件路径: `services/user/internal/handler/user.go`
- 测试结果截图: `docs/screenshots/test_result_20251220.png`
- 验收命令输出:
  ```bash
  $ go test -v ./services/user/...
  PASS
  coverage: 85.2% of statements
  ```
- 部署成功日志: `logs/deployment_20251220.log`
- API 测试结果: `tests/api_test_results.json`
