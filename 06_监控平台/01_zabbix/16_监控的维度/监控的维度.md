# 监控的维度

## 一、zabbix监控总结

### 1、物理硬件监控

```bash
#监控项
cpu温度：温度过高，降频断电保护
风扇转速
主板温度
电压
功率
#监控方式
ipmi工具监控，ipmitool命令行 #真实物理机才能使用
机房巡检 #比如查看硬盘状态，硬盘灯、绿色：正常、黄色：警告、红色：废了
```



### 2、操作系统监控

````bash
#监控项
cpu负载
内存
磁盘容量
磁盘IO
网卡IO
进程数
安全监控/etc/password
#监控方式
zabbix模板，需要优化，不能直接使用
````

```bash
#监控项
nginx
php-fpm
mysql
redis
分布式文件系统glusterFS，ceph

修改开源的模板
```
