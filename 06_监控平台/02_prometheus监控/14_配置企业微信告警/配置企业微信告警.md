# 配置企业微信告警

## 一、创建告警规则

### 1、创建告警规则存放目录

```bash
[root@promethus /prometheus]# mkdir rules
[root@promethus /prometheus]# cd rules
```



### 2、随手写个告警规则

```bash
[root@promethus /prometheus/rules]# vim host-rules.yml

groups:
- name: HostSwapIsFillingUp
  rules:
  - alert: HostSwapIsFillingUp
    expr: (1 - (node_memory_SwapFree_bytes / node_memory_SwapTotal_bytes)) * 100 > 80
    for: 15s
    annotations:
      summary: swap分区满了或者关了
      description: swap分区满了或者关了啊啊啊啊啊啊

```
